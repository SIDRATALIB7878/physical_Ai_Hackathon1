# Capstone: Autonomous Humanoid - Overview

## Learning Objectives

Upon completing this section, the reader will be able to:
- Synthesize knowledge from previous sections to design and implement an end-to-end autonomous humanoid system.
- Integrate ROS 2, simulation, AI perception, navigation, and VLA components into a cohesive system.
- Troubleshoot and validate complex robotic pipelines in simulated environments.
- Articulate the challenges and future directions for fully autonomous humanoid robotics.
